# Shopping Cart

This project is a simple e-commerce website built using React.
It allows users to browse items, add them to a cart, adjust quantities, and view the cart contents dynamically. The interface is clean and responsive, offering a user‑friendly shopping experience.

# Features  
Key features of this shopping cart project include:

- **Add to Cart**: Users can add products to their cart with a single click.  
- **Cart Management**: Items in the cart are listed with their name, price, quantity — users can update quantities or remove items.  
- **Subtotal / Total Price Calculation**: The cart dynamically displays updated totals as items are added or quantities change.  
- **Responsive Design**: The UI adapts to different screen sizes, making it usable on desktop and mobile devices.  
- **Clean UI / UX**: A minimalist layout and intuitive controls for a smooth shopping experience.
